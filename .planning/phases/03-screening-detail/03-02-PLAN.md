方法：
- `loadData()` — 调用 getSymbolList 加载数据
- `onFilter()` — 筛选变化时重置 page=1 重新加载
- `onTableChange(pagination)` — 分页变化时加载
- `resetFilters()` — 清空所有筛选，重新加载
- `customRow(record)` — 返回 `{ on: { click: () => openDrawer(record) } }`
- `rowClassName(record)` — `record.is_holding ? 'holding-row' : ''`
- `trendClass(trend)` — 颜色类名
