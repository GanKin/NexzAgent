```python
def get_symbols(
    self,
    data_date: str = None,
    search: str = None,
    trend_level: str = 'daily',   # daily / weekly / monthly
    trend: str = None,             # 上行趋势 / 下行趋势 / 无趋势
    relative_price: str = None,    # lead / Improving / Weakening / Lag
    leverage: str = None,          # 加杠杆 / 去杠杆
    sort: str = 'relative_strength',
    sort_order: str = 'desc',
    page: int = 1,
    page_size: int = 50,
) -> Dict:
```

**实现逻辑**：

1. 如果 `data_date` 为空，取最新日期（调用 `get_available_dates()[0]`）
2. 构建基础查询：`SELECT * FROM market_overview_data WHERE data_date = %s`
3. 动态拼接 WHERE 条件：
   - `search` → `(symbol ILIKE %search% OR name ILIKE %search%)`
   - `trend` → 根据 `trend_level` 映射到对应列：daily→d_trend, weekly→w_trend, monthly→m_trend
   - `relative_price` → `relative_price_status = %s`
   - `leverage` → `leverage_status = %s`
4. 构建排序：
   - 映射 sort 参数到列名：
     - `relative_strength` → `relative_strength DESC`
     - `early_reversal` → `early_reversal DESC`
     - `leverage_value` → `leverage_value DESC`
     - `trend_duration` → `d_trend_duration ASC`（升序=刚转趋势的排前面）
   - 默认 `relative_strength DESC`
5. 分页：`LIMIT page_size OFFSET (page - 1) * page_size`
6. 先执行 COUNT 查询获取 total，再执行数据查询
7. 对返回的每条记录，添加 `is_holding` 字段（判断 symbol 是否在 HOLDINGS 集合中）
8. 返回：`{ items: [...], total: N, page: N, page_size: N, data_date: str }`

**排序白名单**（防止 SQL 注入）：
```python
SORT_COLUMNS = {
    'relative_strength': 'relative_strength',
    'early_reversal': 'early_reversal',
    'leverage_value': 'leverage_value',
    'trend_duration': 'd_trend_duration',
}
```

**注意**：使用参数化查询，不要用 f-string 拼接用户输入的值。排序字段名从白名单中取，不是直接用用户输入。

```python
def get_symbol_timeline(self, symbol: str, start_date: str = None, end_date: str = None) -> Optional[Dict]:
    """获取单标的时序数据"""
    with get_db_connection() as conn:
        cur = conn.cursor()

        conditions = ["symbol = %s"]
        params = [symbol]

        if start_date:
            conditions.append("data_date >= %s")
            params.append(start_date)
        if end_date:
            conditions.append("data_date <= %s")
            params.append(end_date)

        where = " AND ".join(conditions)

        # 获取标的名称
        cur.execute(f"SELECT DISTINCT name FROM market_overview_data WHERE symbol = %s LIMIT 1", (symbol,))
        name_row = cur.fetchone()
        if not name_row:
            return None
        name = name_row[0]

        # 查询时序数据
        cur.execute(
            f"""
            SELECT data_date::text, d_trend, d_trend_duration,
                   w_trend, w_trend_duration, m_trend, m_trend_duration,
                   relative_price_status, relative_price_duration,
                   leverage_status, leverage_value, leverage_change,
                   relative_strength, price_position_60d
            FROM market_overview_data
            WHERE {where}
            ORDER BY data_date DESC
            """,
            params,
        )
        columns = [desc[0] for desc in cur.description]
        rows = cur.fetchall()
        timeline = [dict(zip(columns, row)) for row in rows]

        return {
            'symbol': symbol,
            'name': name,
            'timeline': timeline,
            'date_range': {
                'start': timeline[-1]['data_date'] if timeline else None,
                'end': timeline[0]['data_date'] if timeline else None,
            }
        }
```
