**区域 5: 持仓快照表（D-11, D-12, D-13, D-15, D-16）**
```vue
<a-card title="持仓快照" size="small">
  <a-table
    :columns="holdingColumns"
    :data-source="holdings"
    :pagination="false"
    size="small"
    row-key="symbol"
  >
    <!-- 趋势列：使用颜色标签 -->
    <template slot="trend" slot-scope="text, record, column">
      <span :class="getTrendClass(text)">{{ text }}</span>
    </template>
    <!-- 变化标记列 -->
    <template slot="change" slot-scope="text">
      <a-tag v-if="text === 'improved'" color="green">↑</a-tag>
      <a-tag v-else-if="text === 'worsened'" color="red">↓</a-tag>
      <a-tag v-else-if="text === 'new'" color="blue">NEW</a-tag>
      <span v-else>-</span>
    </template>
  </a-table>
</a-card>
```

表格列定义：
```javascript
holdingColumns: [
  { title: '代码', dataIndex: 'symbol', width: 80, fixed: 'left' },
  { title: '名称', dataIndex: 'name', width: 100 },
  { title: '日趋势', dataIndex: 'd_trend', width: 90, scopedSlots: { customRender: 'trend' } },
  { title: '周趋势', dataIndex: 'w_trend', width: 90, scopedSlots: { customRender: 'trend' } },
  { title: '月趋势', dataIndex: 'm_trend', width: 90, scopedSlots: { customRender: 'trend' } },
  { title: '比价', dataIndex: 'relative_price_status', width: 100 },
  { title: '杠杆', dataIndex: 'leverage_status', width: 80 },
  { title: '杠杆值', dataIndex: 'leverage_value', width: 80 },
  { title: '60日位', dataIndex: 'price_position_60d', width: 70 },
  { title: '变化', key: 'change', width: 60, scopedSlots: { customRender: 'change' } }
]
```

方法：
- `trendPct(level, type)` — 计算趋势百分比
- `getTrendClass(trend)` — 返回 CSS 类名：'上行趋势' → 'text-green', '下行趋势' → 'text-red', else → 'text-yellow'
- `loadDates()` — 调用 getAvailableDates 获取日期列表
- `loadDashboard(date)` — 调用 getDashboardData 获取数据
- `onDateChange(date)` — 切换日期重新加载

生命周期：`created()` 中调用 `loadDates()` → 自动加载最新日期的 Dashboard 数据
