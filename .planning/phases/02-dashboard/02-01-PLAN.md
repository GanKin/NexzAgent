实现步骤：
1. 查询 `market_overview_data WHERE data_date = %s` 获取当日全量数据
2. 统计趋势分布：COUNT(*) FILTER (WHERE d_trend = '上行趋势') 等（D-04, D-05）
3. 统计资金面：加杠杆/去杠杆数量和百分比（D-06）
4. 统计比价状态：四种状态的数量（D-07）
5. 计算四分类汇总（D-08, D-09, D-10）：

**四分类 SQL 条件（基于 PRD）：**
```sql
-- 主升浪：日=上行 AND 周=上行 AND 杠杆=加杠杆
main_trend: d_trend='上行趋势' AND w_trend='上行趋势' AND leverage_status='加杠杆'

-- 主升调整：(日=上行 OR 周=上行) AND (比价 IN ('Lag','Improving') OR 杠杆=去杠杆)，排除主升浪
main_pullback: NOT main_trend AND (d_trend='上行趋势' OR w_trend='上行趋势') AND (relative_price_status IN ('Lag','Improving') OR leverage_status='去杠杆')

-- 新信号/反转：三选一条件，排除以上两类
new_signal: NOT main_trend AND NOT main_pullback AND (
    (d_trend='上行趋势' AND d_trend_duration <= 5) OR
    (d_trend='上行趋势' AND relative_price_status='Improving' AND leverage_status='加杠杆') OR
    (early_reversal > 110 AND relative_strength > 95 AND leverage_status='加杠杆')
)

-- 规避区：日=下行 AND 周=下行 AND 杠杆=去杠杆
avoid: d_trend='下行趋势' AND w_trend='下行趋势' AND leverage_status='去杠杆'
```

```python
@market_overview_bp.route('/dashboard', methods=['GET'])
@login_required
@admin_required
def get_dashboard():
    """获取 Dashboard 全量数据（D-22, D-23, D-24）"""
    date = request.args.get('date', None)

    if not date:
        # 默认取最新日期
        dates = _service.get_available_dates()
        if not dates:
            return jsonify({'code': 0, 'msg': '暂无数据', 'data': None}), 200
        date = dates[0]

    try:
        data = _service.get_dashboard_data(date)
    except Exception as e:
        logger.error(f"Dashboard 数据查询失败: {e}", exc_info=True)
        return jsonify({'code': 0, 'msg': f'查询失败: {str(e)}', 'data': None}), 500

    return jsonify({
        'code': 1,
        'msg': 'success',
        'data': data,
    }), 200
```

关键点：
- `date` 参数可选，默认使用最新日期（调用已有的 `get_available_dates()`）
- admin-only 保护（@login_required + @admin_required）
- 错误处理：数据库异常返回 500，无数据返回空结果
</action>
