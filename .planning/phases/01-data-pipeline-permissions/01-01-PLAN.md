**关键实现细节**：
- `import_data` 的 UPSERT 使用：
  ```sql
  INSERT INTO market_overview_data (data_date, symbol, name, data_type, ...)
  VALUES (%s, %s, %s, %s, ...)
  ON CONFLICT (data_date, symbol) DO UPDATE SET
      name = EXCLUDED.name,
      relative_strength = EXCLUDED.relative_strength,
      ...
      updated_at = NOW()
  ```
- 跟踪新增 vs 更新：先查询已有 (data_date, symbol) 对比，或使用 `RETURNING xmax`（PostgreSQL xmax trick: xmax=0 表示 insert, >0 表示 update）
