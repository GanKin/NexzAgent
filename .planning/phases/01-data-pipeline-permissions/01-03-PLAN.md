**2. 创建 `gversion/QuantDinger-Vue/src/api/marketOverview.js`**：

```javascript
import request from '@/utils/request'

const api = {
  upload: '/api/market-overview/upload',
  uploads: '/api/market-overview/uploads',
  dates: '/api/market-overview/dates'
}

export default api

/**
 * 上传 Excel 文件
 * @param {File} file - Excel 文件对象
 * @returns {Promise} 导入摘要
 */
export function uploadMarketData (file) {
  const formData = new FormData()
  formData.append('file', file)
  return request({
    url: api.upload,
    method: 'post',
    data: formData,
    headers: { 'Content-Type': 'multipart/form-data' },
    timeout: 30000 // 30秒超时（大文件上传）
  })
}

/**
 * 获取上传历史
 */
export function getUploadHistory (parameter) {
  return request({
    url: api.uploads,
    method: 'get',
    params: parameter
  })
}

/**
 * 获取可用日期列表
 */
export function getAvailableDates (parameter) {
  return request({
    url: api.dates,
    method: 'get',
    params: parameter
  })
}
```

**2. `gversion/QuantDinger-Vue/src/views/market-overview/UploadCard.vue`**：

上传组件（D-09, D-10, D-11）：
- 使用 Ant Design Vue 的 `a-upload-dragger` 组件实现拖拽+点击上传
- 限制单文件上传，只接受 `.xlsx` / `.xls`
- 上传中显示 loading 状态和进度条
- 上传成功后显示导入摘要卡片：
  ```
  数据日期: 2026-05-18
  数据类型: 美股市场
  总行数: 1000
  新增: 850 | 更新: 150
  ```
- 上传失败时显示错误详情（行号 + 字段 + 原因）
- 上传前验证文件类型和大小（≤10MB）

```vue
<template>
  <a-card title="上传数据" :bordered="false">
    <a-upload-dragger
      name="file"
      :multiple="false"
      :action="uploadUrl"
      :headers="uploadHeaders"
      accept=".xlsx,.xls"
      :before-upload="beforeUpload"
      @change="handleUploadChange"
    >
      <p class="ant-upload-drag-icon">
        <a-icon type="inbox" />
      </p>
      <p class="ant-upload-text">点击或拖拽 Excel 文件到此区域上传</p>
      <p class="ant-upload-hint">支持 .xlsx 格式，单次上传一个文件</p>
    </a-upload-dragger>
    
    <!-- 上传成功摘要 -->
    <a-alert v-if="uploadResult" type="success" show-icon style="margin-top: 16px">
      <template slot="message">
        <span>导入成功</span>
      </template>
      <template slot="description">
        <div>数据日期: {{ uploadResult.data_date }}</div>
        <div>数据类型: {{ uploadResult.data_type }}</div>
        <div>总行数: {{ uploadResult.total_rows }} (新增: {{ uploadResult.new_rows }}, 更新: {{ uploadResult.updated_rows }})</div>
      </template>
    </a-alert>
    
    <!-- 上传失败 -->
    <a-alert v-if="uploadError" type="error" show-icon style="margin-top: 16px">
      <template slot="message">上传失败</template>
      <template slot="description">
        <div>{{ uploadError }}</div>
      </template>
    </a-alert>
  </a-card>
</template>
```

关键实现细节：
- `uploadUrl` 从 api 文件获取
- `uploadHeaders` 包含 `Authorization: Bearer ${token}`
- `beforeUpload` 验证文件类型和大小
- `handleUploadChange` 处理上传状态变化，成功时 emit `upload-success` 事件
- 不使用 `a-upload-dragger` 的 `action` 属性直接上传，改为手动调用 `uploadMarketData` API 以便控制流程

**样式**：
- 使用 Ant Design Vue 默认样式，不过度自定义
- 保持与其他页面一致的布局风格
- 响应式布局：桌面端双列，移动端单列
</action>
